### sử dụng using namespace std;

vì phải nhập quá nhiều dòng std:: nên chúng ta mặc định rằng sẽ sử dụng namespace std cho toàn bộ chương trình.

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	cout << "Hello World!"<< endl;
	// chương trình sẽ xuất ra dòng chữ "Helloo World!"
}
```
